Hagamos Ciencia con la Basura en las Playas de México
Este es un proyecto de desarrollo web diseñado para la asociacion Ocean conectors que busca concientizar y recolectar datos sobre la contaminación por residuos en las costas mexicanas. La aplicación permitira a los niños registrar diferentes tipos de materiales encontrados en las playas, visualizando el impacto ambiental a través de gráficas interactivas.

Características Principales
Registro de Residuos: Formulario intuitivo para capturar fecha, ubicación y tipo de material.

Visualización de Datos: Gráficas de barras dinámicas que muestran los totales de materiales recolectados.

Filtros Inteligentes: Capacidad de segmentar la información por Mes o por país para analizar tendencias regionales.

Diseño Responsivo: Interfaz adaptada para dispositivos móviles con un menú de navegación optimizado.

Identidad Visual: Uso de ilustraciones amigables y una paleta de colores vibrante para fomentar la participación de los niños.

Tecnologías Utilizadas:
React: Biblioteca principal para la construcción de la interfaz.

Vite: Herramienta de construcción para un desarrollo rápido.

CSS3: Animaciones personalizadas, Media Queries para diseño responsivo y efectos visuales (text-shadow, box-shadow).

JavaScript (ES6+): Lógica de filtrado de datos y manejo de estados con useState y useMemo.
